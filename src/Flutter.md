# Flutter

## Structure

lib/ -> Source code of app.
- localizations/  -> Grouping localizations when app is for multiples languages
- routes/         -> Contains apps pages
- models/         -> Is for the "business logic"
  - daos/
  - repositories/
  - services/
  - etc...
- widgets/        -> Reusable UI Widgets
- main.dart
- routes.dart
